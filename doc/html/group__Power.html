◆ acc_service_power_bins_power_bins_callback_set()

void acc_service_power_bins_power_bins_callback_set ( acc_service_configuration_t  service_configuration,
acc_power_bins_callback_t *  power_bins_callback,
void *  user_reference 
)

Set a callback to receive power bins results.

If a callback is used, power bins results are indicated by calling the function that is set. Within the callback it is only allowed to copy the data to a application memory to allow the best possible service execution. Setting the callback as NULL disables callback operation.

Parameters
[in]service_configurationThe configuration to set a callback for
[in]power_bins_callbackThe callback function to set
[in]user_referenceA user chosen reference that will be provided when calling the callback
