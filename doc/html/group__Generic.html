void acc_service_destroy ( acc_service_handle_t *  service_handle)

Destroy a service identified with the provided service handle.

Destroy the context of a service allowing another service to be created using the same resources. The service handle reference is set to NULL after destruction.

Parameters
[in]service_handleA reference to the service handle to destroy
